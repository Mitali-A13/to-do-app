A Simple To-Do App
A Progressive Web App for managing your daily tasks.

Live App
https://bucolic-caramel-e7d363.netlify.app

Features

Add, edit, and delete tasks
Works offline
Install on any device
Data saved locally

Install on Your Device
iPhone:
Open in Safari → Share → Add to Home Screen
Android:
Open in Chrome → Menu → Install app
Desktop:
Open in Chrome/Edge → Click install icon in address bar

Technologies
HTML, CSS, JavaScript, Service Workers
